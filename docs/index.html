AWS iGenomes

Common reference genomes hosted on AWS S3

This website hosts some bash scripts which can be directly run to easily obtain reference genomes where you need them. Note that they all require you to have the aws command line tools installed and set up with proper authentication.

For more information about this resource, please see the GitHub readme at https://github.com/ewels/AWS-iGenomes.

Generic downloads:

Script source: https://github.com/ewels/AWS-iGenomes/blob/master/docs/aws-igenomes.sh

Script address: https://ewels.github.io/AWS-iGenomes/aws-igenomes.sh

Quick usage:


    curl -fsSL https://ewels.github.io/AWS-iGenomes/aws-igenomes.sh | bash

Advanced usage:


    curl -fsSL https://ewels.github.io/AWS-iGenomes/aws-igenomes.sh > aws-igenomes.sh
    bash aws-igenomes.sh --help
